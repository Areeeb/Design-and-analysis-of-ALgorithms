- Introduction to Algorithms by Thomas cormen, Charlese Leiserson, Ronaldl. Rivest, Clifford stein
- Algorithm Design by Jon Kleinberg, Eva Tardos
- Algorithms by Dasgupta, C.H papadimitriou and U.V. Vazirani
